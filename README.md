# sistema-loja-estoque-java
O sistema foi desenvolvido em Java utilizando Programação Orientada a Objetos.

A arquitetura foi separada em:
model, service, controller e main.

O projeto possui:
cadastro de produtos, clientes, vendas,
controle de estoque, controle de caixa,
login com níveis de acesso,
emissão de cupom fiscal
e relatórios de vendas e lucro.

Também foi aplicada herança através da classe
Funcionario que herda de Usuario.
